# DocAnon — Локальный анонимизатор документов

> Обезличивание персональных данных перед отправкой в AI-сервисы.
> Всё работает на вашем ПК. Ничего не покидает машину.

## Зачем

По 152-ФЗ запрещено отправлять персональные данные в зарубежные сервисы.
ChatGPT для проверки договора = нарушение закона. Этот сервис решает проблему:
документ обезличивается локально, а чистый текст можно безопасно отправлять в любой AI.

## Алгоритм анонимизации

### Шаг 1: Извлечение текста
- PDF → pdfplumber (текстовые) / PyMuPDF → Tesseract (сканы)
- DOCX → python-docx (параграфы + таблицы)
- TXT → прямое чтение
- Images → Tesseract OCR

### Шаг 2: Поиск персональных данных (многослойный)

**Слой 1 — Regex (мгновенно, 100% точность для паттернов):**
- ИНН (10/12 цифр, валидация чексуммой)
- СНИЛС (XXX-XXX-XXX XX, валидация)
- Телефон (+7/8, 11 цифр)
- Паспорт (4+6 цифр)
- ОГРН (13 цифр, валидация)
- Email
- Банковские реквизиты

**Слой 2 — GLiNER NER (контекст, ~95% recall):**
- PERSON — люди
- ORG — организации
- LOC — адреса, города
- Custom labels: ADDRESS, DATE_OF_BIRTH

**Слой 3 — pymorphy3 (дополнительная проверка):**
- Имена в любом падеже/роде/числе
- Отсечение ложных срабатываний NER

**Слой 4 — LLM (опционально, для сложных случаев):**
- Контекстные зависимости ("контактное лицо Иванов И.И.")
- Когда NER не уверен (confidence < порога)

### Шаг 3: Замена
- Каждая сущность → плейсхолдер: `[ФИО]`, `[ИНН]`, `[ТЕЛЕФОН]`, `[АДРЕС]`, `[ОРГАНИЗАЦИЯ]`
- Сохранение маппинга для обратного восстановления (опционально)
- Оригинал НЕ сохраняется (по умолчанию)

### Шаг 4: Экспорт
- Текст с плейсхолдерами
- DOCX с сохранением форматирования
- JSON со статистикой (сколько сущностей, типы, confidence)

## Режимы работы

1. **Быстрый** — только regex + GLiNER (секунды)
2. **Точный** — regex + GLiNER + pymorphy3 + LLM (минуты)
3. **Ручной** — показывает все найденные сущности, пользователь подтверждает

## Конфигурация

Пользователь может включать/выключать:
- Типы сущностей (ФИО, ИНН, телефоны, адреса, etc.)
- Алгоритмы (regex / NER / LLM)
- Порог уверенности NER
- Формат плейсхолдеров (стандартный / кастомный)
- Язык интерфейса (RU / EN)
